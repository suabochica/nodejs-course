:spider: Web Scrapper
========================

Scraping es la actividad de leer un contenido y extraer información específica del mismo. En este proyecto se busca consular la disponibilidad del la consola Xbox Series X en diferentes proveedores: Amazon, Microsoft, fnac, Game, etc.

🧰 Tool Kit
-----------

La siguiente lista recopila las tecnologías utilizadas en este proyecto.

- [Playwright](https://playwright.dev/docs/intro#installing-playwright), para acomodar las necesidades de las pruebas end-to-end a través de automatización de tareas


🧞 Comandos
-----------

Todos los comandos se ejecutan en la raíz del proyecto

| Command                    | Action                                           |
| :------------------------- | :----------------------------------------------- |
| `pnpm install`             | Instala dependencias                             |
| `pnpm run dev`             | Ejecuta el programa de scrapping                 |
